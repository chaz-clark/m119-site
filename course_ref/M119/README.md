# M119

Course materials for 4-credit M-119 Applied Calculus for Data Analysis

## Outcomes

1.	Apply their understanding of functions across discrete and continuous probability distributions and algebraic functions as mathematical models.
2.	Communicate mathematics using correct mathematical notation and appreciate the power of mathematical notation’s role in data analytics.
3.	Use symbolic and numerical methods of differentiation to optimize functions and find the linear approximation of functions.
4.	Use symbolic and numerical methods of integration to calculate probabilities using probability density functions and cumulative distribution functions and derive expected value and variance of functions of random variables.

## Prerequisites

Pre-Requisite: (CS 101, CSE 110 or CS 124, CIT 160) and (ACT 23 or SAT 540 (Post 2016, 570)) or ALEKS 55 or MATH 101 or MATH110X or MATH109 or FDMAT112)
